## Frobenius Norm

![img_6.png](img_6.png)

- L2 노름과 비슷하게, 다차원 행렬에서 거리를 의미할 수 있어

```python
import numpy as np
import torch
import tensorflow as tf

X = np.array([1,2],[3,4])
np.linalg.norm(X)
# torch, tf 에선 float 타입으로 넣어줘야 작동해. 
torch.norm(X)
tf.norm(X)


```

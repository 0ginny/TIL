# Branch 소개

타임라인 같은 것.

한 브랜치에서 작업을 할 때 다른 브랜치에 영향이 없어.

별도로 존재하는 것.

# HEAD 란?

내 마지막 작업, 북마크?

정확히는 Branch Point

# 명령어

### 브랜치 보기
- git branch
  - see all branch and my head(*)

### 브랜치 만들기
- git branch BranchNameIWant
  - 기억할 건 , 만들었다고 바로 그 브랜치로 가는 것이 아니야.
  - 헤드를 변경해줘야해.

### 헤드 바꾸기
- git switch 목표브랜치

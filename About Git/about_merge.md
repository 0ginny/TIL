# Merge

병합은 브랜치를 병합하는 거야.

특정 커밋과 커밋을 병합하는 것이 아니야

항상 HEAD를 기준으로 병합해.

그래서 병합을 하기 전에 반드시 원하는 곳으로 switch한 후에 병합해야해.

## 병합 종류
- 연결된 브랜치에서 병합시에는 그저 헤드 포인터를 바꾸기만 하면 돼.
  - fast forward merge

- 만약 여러 브랜치로 나누고 작업이 되었을 경우. 
  - 병합 커밋  - 부모를 두개 가짐  

# 명령어

- git merge 병합할브랜치
  - 이전에 미리 기준이 될 브랜치로 이동해야해.
